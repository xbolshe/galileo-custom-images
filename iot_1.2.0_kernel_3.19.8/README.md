Brief
====
This SD card Linux image for Galileo Gen1/Gen2 boards is based on:
* Base: [BSP 1.2.0](https://downloadcenter.intel.com/download/23197/Intel-Quark-BSP)
* Kernel: 3.19.y (3.19.8 on 12.12.2015)
* Many packets shown below

====
